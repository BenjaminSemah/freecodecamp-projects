Introduction

This page is a compilation of ten online resources that I have found helpful in my journey as a developer. And I’ll explain why you should also consider using them too.

They include developer communities, podcasts, books, and more. And all of these resources are beginner friendly.

FreeCodeCamp

FreeCodeCamp is one of the popular coding websites on the internet. The curriculum is well-structured and beginner friendly. Many boot camps incorporate FreeCodeCamp lessons into their curriculum. The FreeCodeCamp community is also one of the most vibrant community of programmers online.

FreeCodeCamp is a whole package and includes the following.

  • The FreeCodeCamp Website (learning platform)
  • The FreeCodeCamp YouTube Channel
  • The FreeCodeCamp Publication
  • The FreeCodeCamp Podcast
CodeNewbie Podcast

When I started learning to code, I often googled “podcast recommendations for new programmers”. One that caught my attention always was The CodeNewbie Podcast. It appeared either at the top or close to the top of all the search results I clicked on. I decided to check it out. And it sure did live up to the hype.

One thing I love about The CodeNewbie Podcast is the diverse backgrounds of the guests. There’ve been people on the podcast who are;

  • Computer Science graduates
  • Musicians
  • Truck drivers
  • Stay-at-home moms
  • And more

Saron Yitbarek, the host, talks to these amazing people “on their coding journey in hopes of helping you on yours”. If you’re a code newbie, check out the CodeNewbie Podcast.

Scrimba Podcast

Alex Booker hosts the Scrimba podcast which features interviews of both recently hired developers and industry experts as guests. A new episode drops every Tuesday.

Get A Dev Job

The title of the book is “How to Learn to Code and Get a Developer Job”. It’s written by Quincy Larson, founder of FreeCodeCamp. This book contains practical advice on how to increase your chances of landing your first developer job. Solid advice in there. No fluff or BS.

Build A Portfolio

This short e-book by Josh Comeau shows you the ropes of building a dev portfolio that is appealing to both recruiters and other developers. If you’re yet to land your first job as a developer, at some point it would be useful for you to have a personal website or portfolio. A place where you have some of your projects on display.

Hashnode

“Hashnode is everything you need to start blogging as a developer.” If you’re a tech writer, looking for a supportive community of like-minded fellows, Hashnode is the place to be.

NoCSDegree

If you’re working on breaking into tech and you don’t have a Computer Science degree, you should check out the NoCSDegree website. As the name suggests, this is a website that features stories of people who started their tech careers with no Computer Science (CS) degree.

You probably know already that you don’t need a CS degree to land a job in tech. But it still helps if someone points to people who have done it. That’s what Pete, who runs NoCSDegree does with these articles. The interview-style articles are often short and can be read in about five minutes.

First Timers Only

First Timers Only Contributing to open source for the first time can be scary and a little overwhelming. Perhaps you’re a Code Newbie or maybe you’ve been coding for a while but haven’t found a project you felt comfortable contributing to. First Timers Only will teach you how to contribute to open-source projects.

PRAMP

PRAMP is a website that pairs you with other developers. You take turns as a pair to act as interviewer and interviewee. PRAMP provides the questions that you would use for the mock interviews - both technical and behavioral interviews.

MDN Docs

The MDN Docs include documentation for web technologies including the following.

HTML CSS JavaScript Python Accessibility

I use this website as a reference source. When I need to look up more information about a particular topic, it's one of the first places I go to.

There’s an IDE on the website that you can use to write and run example code. This is useful because as you read the documentation, your can practice or play around with the code to get a better understanding of the topic you’re reading about. This is one of those websites you should bookmark and visit often.
